# Anka-Docs Template

Bootstrap kullanılarak geliştirilen belgeler temasıdır.

## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
git clone https://github.com/kayadev/Anka-Docs.git
```

Proje dizinine gidin

```bash
cd my-project
```

Gerekli paketleri yükleyin

```bash
npm install
```

Sunucuyu çalıştırın

```bash
npm run compile:sass
```
